# Vue 3 + Vite

## Установка проекта
Склонируйте репозиторий:

```bash
git clone https://github.com/yato90/my_tasks.git
cd my_tasks
```

Установка зависимостей:

```bash
npm install
```

Запуск json сервера:

```bash
start json-server --watch db.json --port 3000
```

Запуск проекта

```bash
npm run dev
```
## Просмотр
Перейти по: http://localhost:5173
для json: http://localhost:3000
